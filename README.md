# react-native-pdf-viewer

PDF viewer for React Native

## Installation

```sh
npm install react-native-pdf-viewer
```

## Usage

```js
import PdfViewer from "react-native-pdf-viewer";

// ...

const result = await PdfViewer.multiply(3, 7);
```
